### Example Dataset Details
The dataset consist of three files in zipped format where one is VCF file in .vcf format, second one is VCF file in  .vcf.gz format and third one is tabixed VCF file in vcf.gz.tbi format. Users are suggested to use those .vcf.gz and .vcf.gz.tbi formatted files as input after extraction. To put the Chromosome No, Start Position and End position of the chromosome users are suggested to view the VCF file in .vcf format only.
