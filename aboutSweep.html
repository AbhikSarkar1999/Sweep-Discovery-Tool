Sweep Discovery Tool

Background

Selective Sweep is a biological phenomenon in which genetic variations among neighbouring the beneficial mutant allele are swept away due to the effect of genetic hitchhiking.Selective sweep has vast importance in genomic studies related to phylogeny, development of new varieties, disease and pathogen genomics etc.

About Sweep Discovery Tool

“Sweep Discovery Tool” is a user friendly tool for detecting selective sweep in genomic regions. It uses Random Forest based machine learning approach to predict selective sweep from VCF files as an input.

Instructions for Users

  1. File Format: Provide the input file in .vcf.gz (gzipped VCF format) and .vcf.gz.tbi (tabixed VCF format). Please use valid VCF files with proper information of samples.
  2. Input Parameters:Provide input files along with other mandatory input parameters such as Chromosome No, Starting Position and Ending Position of the region of interest of the concerned Chromosome.
  3. Input Details: After providing all the input files save them in a directory by "Save" button and then put other parameters and run the "Predict" button for selective sweep prediction.
